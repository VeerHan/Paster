# 贡献指南

感谢你对 Paster 项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

- 在 [Issues](../issues) 中搜索是否已有类似问题
- 若无，请新建 Issue，并尽量包含：
  - 系统版本（macOS 版本号）
  - 复现步骤
  - 预期行为与实际行为

### 提交代码

1. **Fork** 本仓库
2. 创建分支：`git checkout -b feature/your-feature` 或 `fix/your-fix`
3. 遵循现有代码风格（Swift 推荐使用 [Swift API Design Guidelines](https://swift.org/documentation/api-design-guidelines/)）
4. 确保项目可以正常编译运行
5. 提交并 Push 到你的 Fork
6. 在本仓库创建 [Pull Request](../pulls)，简要描述改动内容

### 代码风格

- 使用 Swift 官方命名规范
- 适当添加注释，尤其是公开 API
- 保持单文件职责清晰

### 开发环境

- Xcode 15.0+
- macOS 13.0+

再次感谢你的贡献！
